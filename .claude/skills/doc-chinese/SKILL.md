# 文档写作 — 强制汉语输出

> **核心规则：凡涉及文档写作，一律使用汉语（中文）输出。**

## 触发条件

当用户的请求中包含以下关键词或意图时，本 skill 自动触发：

- 写文档 / 写个文档 / 帮我写文档
- 写报告 / 写总结 / 写方案
- 写README / 写说明 / 写纪要
- 写技术文档 / 设计文档 / 需求文档
- 写笔记 / 写会议纪要
- 任何要求生成、创建、编写文档类内容的请求

## 行为规则

1. **语言**：文档正文必须使用汉语（简体中文）书写
2. **术语**：技术术语可保留英文原文，但需用中文加以解释
3. **格式**：文档采用 Markdown 格式，标题层级清晰
4. **风格**：使用正式、专业的书面汉语，避免口语化表达
5. **代码块**：代码示例不受语言限制，注释使用中文

## 示例

### ✅ 正确

```markdown
# 系统架构设计文档

## 概述
本系统采用微服务架构，主要包含以下模块...

## 技术选型
- **Spring Boot**：作为核心框架，提供依赖注入和自动配置能力
```

### ❌ 错误

```markdown
# System Architecture Design Doc

## Overview
This system adopts microservice architecture...
```

## 例外情况

- 用户明确指定使用其他语言时，按用户要求执行
- 代码注释和变量名等编程元素不受此限制
